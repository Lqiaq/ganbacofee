# Ganba Coffee — weboldal sablon

Egyszerű, statikus egyoldalas weboldal a **Ganba Coffee & Space** (Debrecen, Simonffy utca 4-6.) számára.

## Jellemzők
- Reszponzív, egyfájlos statikus oldal (`index.html`)
- Magyar / angol nyelvváltó (HU/EN)
- Nyitvatartás, kapcsolati adatok, beágyazott Google-térkép
- Szezonális kínálat és galéria a kávézó képeivel

## Helyi megnyitás
Nyisd meg az `index.html` fájlt böngészőben — nincs szükség build lépésre.

## Deploy (Vercel)
1. Importáld ezt a repót a [vercel.com](https://vercel.com) felületén.
2. Framework preset: **Other** (nincs build parancs).
3. Deploy — az `index.html` lesz a főoldal.
